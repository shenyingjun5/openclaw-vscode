# 待办事项

- [ ] **自动重连机制**
  - 当前：WebSocket 断开后需要手动重连
  - 目标：实现自动重连（带指数退避）
  - 优先级：高（影响用户体验）

- [ ] **Agent 删除功能完善**
  - 当前：删除 Agent 需要通过 RPC，Gateway 未连接时无法删除

  - 优先级：高（功能完整性）

- [ ] **错误恢复机制**
  - 当前：某些错误会导致状态不一致
